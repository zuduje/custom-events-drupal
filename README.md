# 📱 custom_events — Módulo Drupal 10

Módulo personalizado de Drupal 10 para gestión de eventos e inscripciones de usuarios, desarrollado bajo la línea gráfica de **Móvil Éxito**.

---

## 🚀 Funcionalidades

- ✅ Crear eventos con título, descripción, país y fecha
- ✅ Países obtenidos desde [restcountries.com](https://restcountries.com)
- ✅ Listado de eventos con diseño Móvil Éxito
- ✅ Inscripción AJAX sin recargar la página
- ✅ Validación de doble inscripción
- ✅ Contador de inscritos por evento
- ✅ Control de acceso por roles

---

## 🛠️ Tecnologías

| Capa | Tecnología |
|------|------------|
| Backend | PHP 8.4 + Drupal 10 |
| Frontend | Twig + CSS + JavaScript vanilla |
| Base de datos | MySQL 8.0 |
| Servidor | Apache 2.4 + Ubuntu 24.x |
| Estilos | CSS Variables + Nunito Sans |
